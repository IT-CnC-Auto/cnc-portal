The three agents working the GBP profiles

Each one runs on its own schedule, writes to audit_event, surfaces here
AGENT 19 · AUDITOR
GBP Auditor

Runs every 14 days at 03:00 SAST against all four profiles. Scores each one against Google's 16-point checklist. Writes a gbp_score row per location, a gbp_action row per gap, a one-page PDF audit report on CNC letterhead.

  • Model: Anthropic Sonnet 4.6
  • Rand per audit: R 0.85
  • Output: gbp_score + gbp_action rows + PDF
AGENT 20 · OPTIMISER · AUTOPILOT
GBP Optimiser

Reads the open gbp_action queue every hour. Drafts the change. Writes it. Posts it. Twice-weekly Google Posts. Twelve-hour review-reply target on all incoming reviews. Weekly photo refresh. Every action writes an audit_event row and a Resend digest to Tanya Nel at end of day.

  • Model: Anthropic Sonnet 4.6 (Helena voice locked)
  • Rand per action: R 0.20 to R 1.50
  • Autopilot scope: posts, 5★ replies, photo captions, attributes
  • Human approval: 1-3★ replies, business description, new services
AGENT 21 · ENGAGEMENT
GBP Engagement Analyst

Reads gbp_score history, customer-action data, review velocity. Adapts the post cadence per location based on what is actually working. Writes recommendations to ad_recipe for the rest of the marketing programme. Quantifies time saved in rand each month.

  • Model: Anthropic Sonnet 4.6
  • Frequency: weekly
  • Output: recipe + avoid rows + Director email

FLOOR
✓
Anomaly detection · daily Director digest

ALERT
✓
One-click kill-switch
The PAUSE AUTOPILOT button at the top of this page halts all three GBP agents within 60 seconds. State is preserved. Resume requires both Director signatures via Resend magic-link.
KILL
✓
Append-only audit log
Every action writes an audit_event row with prompt hash, output hash, rand cost, latency, and the human reviewer if approval was required. Twelve-month retention by default, indefinite for Director-flagged events.
AUDIT
✓
Daily rand ceiling: R 25 per location
If a single location burns more than R 25 of agent budget in 24 hours, autopilot pauses for that location and asks a human to look. Protects against runaway prompt loops.
